### 流程图
> TB - top bottom（自上而下）   
> BT - bottom top（自下而上）  
> RL - right left（从右到左）  
> LR - left right（从左到右）  
> 参考：[使用说明](http://knsv.github.io/mermaid/#graph)
```
graph TD
A[矩形]-->B(圆角矩形)
B[矩形]-->C{菱形}
C --> |实线| D((圆形))
C --第二种写法--> E((圆形))
C -.->|点线| F((圆形))
C ==> |粗线|G((圆形))
```
